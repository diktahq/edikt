# /edikt:team (deprecated)

This command has been merged into [`/edikt:init`](/commands/init) and [`/edikt:config`](/commands/config).

When you run `/edikt:init` on a project that already has edikt configured, it validates your local environment — git identity, Claude Code version, MCP environment variables, hooks, and edikt version compatibility. This is the flow that `/edikt:team setup` used to provide.

For viewing and modifying project configuration, use [`/edikt:config`](/commands/config).

This stub will be removed in v0.5.0.
