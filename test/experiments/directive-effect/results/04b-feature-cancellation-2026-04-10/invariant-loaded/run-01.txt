### One-review-per-user enforcement

`Create` calls `GetByUserAndProduct` first. If a row exists, it returns `ErrAlreadyReviewed` → HTTP 409. The handler maps each service error to its appropriate status code (`ErrInvalidRating` → 422, `ErrAlreadyReviewed` → 409).

### Migration needed

The `reviews` table doesn't exist yet. You'll need:

```sql
CREATE TABLE reviews (
    id         TEXT PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id    TEXT        NOT NULL,
    tenant_id  TEXT        NOT NULL,
    product_id TEXT        NOT NULL,
    rating     SMALLINT    NOT NULL CHECK (rating BETWEEN 1 AND 5),
    body       TEXT        NOT NULL DEFAULT '',
    created_at TIMESTAMPTZ NOT NULL,
    UNIQUE (user_id, product_id, tenant_id)
);
CREATE INDEX ON reviews (product_id, tenant_id);
```
